# Shadow Inventory Dictionary

## Fields

- `date`: Monthly observation date in `YYYY-MM-DD` format.
- `shadow_inventory_index`: Synthetic inventory pressure index used for demo modeling.
- `borough`: Borough label tied to the observation.

## Notes

- This is a demo dictionary used to exercise source-object attachments in the Data Workbench.
- In a fuller project, this file would include source lineage, units, missingness notes, and update caveats.
