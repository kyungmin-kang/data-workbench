# Data Workbench Codex Plugin

This is a repo-local Codex plugin scaffold for the Data Workbench.

It is intentionally thin:

- skills wrap the existing workbench role contracts
- playbook detail stays in `docs/agent_playbooks/`
- the app remains the source of truth

Included skills:

- `workbench-architect`
- `workbench-scout`
- `workbench-builder`
- `workbench-qa`

The plugin is optional for the contributor-ready public GitHub release. The workbench remains fully usable without it.
